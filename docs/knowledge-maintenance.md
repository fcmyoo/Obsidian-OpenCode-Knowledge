# Knowledge Maintenance

> 固定节奏地维护知识库，减少“想起来再整理”的摩擦。

## 1. 触发词与频率

| 频率 | 触发词 | 动作 |
|------|--------|------|
| 每日 | 加到 wiki / 爬了这个 | 录入 raw，编译 wiki |
| 每周 | lint wiki | 检查索引、链接、孤岛 |
| 每月 | 月度整理 | 更新 index，合并重复标签 |
| 季度 | 季度重构 | 合并相关主题，删除过时内容 |

## 2. 周检清单

- `wiki/index.md` 和实际文件一致
- 内部链接无失效
- 孤岛页面低于阈值
- `log.md` 连续更新

## 3. 月检清单

- 标签体系收敛
- 重复内容识别
- 主题边界检查
- 用户自定义规则回顾

## 4. 季检清单

- 知识域分类有效性
- 长期未访问内容归档
- 结构复杂度评估
