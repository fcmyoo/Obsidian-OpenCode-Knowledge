# Plugin Recommendations

> 在现有 skills 基础上，按需补充 Obsidian 插件。

## 1. Dataview

适合：动态查询、筛选列表。
注意：适合只读视图；修改知识库内容仍按 AGENTS 规则处理。

## 2. Tag Wrangler

适合：批量重命名、合并标签。
触发：每月标签整理。

## 3. Periodic Notes

适合：日记、周记模板。
注意：`wiki/log.md` 仍是唯一操作日志。

## 4. 使用边界

- 插件做展示层
- 规则和流程仍由 AGENTS/AI_CONFIG 控制
- 避免插件和 rules 重复定义同一逻辑
