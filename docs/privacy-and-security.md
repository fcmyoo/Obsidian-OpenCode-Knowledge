# Privacy and Security

> 本地知识库的数据边界说明。

## 1. 数据存储

- 所有数据保留在本地
- 普通 Markdown 文件，可随时备份
- 不强制上传云端

## 2. AI 处理边界

- raw 原始素材不会被 AI 修改
- AI 整理时，默认不把内容发往外部存储
- 仅当前配置的 AI provider 会收到内容用于生成

## 3. 外部链接

- 外部 URL 仅保留在 raw frontmatter
- wiki 的 Sources 只链接本地 raw 文件

## 4. 分享注意

- 分享前检查 frontmatter 是否含敏感信息
- assets 路径不暴露账户凭据
